Box Component


  • This component can be used to display values and or trends.
  • panelClass sets the overall color (see BootStrap 4.0 colors
  • icons are based on FontAwesome (see https://fontawesome.com)

	
		<div id="box1"></div>	
		...
		new BoxComponent({
			title:"Total Revenue",
			targetId:"box1",
			action:"Revenue Dashboard",
			actionClick: "window.location.href ='/views/examples/boxcomponent.html'",
			icon:"fa fa-shopping-cart fa-5x",
			value:"2 Million",
			panelClass:"overview-item--c1"
		}).draw();

Box Chart Component

  • The box chart component adds a timeseries graph and calculates the values based on the data provided.
  • All chart components allow you to pass in an ADQL query instead of the data property. If you pass in both, the query would be executed first.
  • if you remove the data and query properties then the component will generate random or test data.
  • The query and data should provide timeseries data for the timeseries chart.

The following example we provide the data.

			
					var chartData =[
						[1532966193039,5],
						[1532966013039,10],
						[1532965833039,20],
						[1532965593039,7],
						[1532965293039,30]
					];
				
					new BoxChartComponent({
						title:"Total Revenue",
						targetId:"box2",
						action:"Revenue Dashboard",
						actionClick: "window.location.href ='/views/examples/boxcomponent.html'",
						icon:"zmdi zmdi-shopping-cart",
						data:chartData,
						//query:,
						options:{
							size:{height:75,width:250}
						},
						panelClass:"overview-item--c2"
					}).draw();

The following example since we did not provide a query or data property, the chart generates its own data.

			
					new BoxChartComponent({
						title:"Total Revenue",
						targetId:"box3",
						action:"Revenue Dashboard",
						actionClick: "window.location.href='/views/examples/boxcomponent.html'",
						icon:"fa fa-shopping-cart fa-2x",
						//data:,
						//query:,
						options:{
							size:{height:75,width:250}
						},
						panelClass:"overview-item--c3",
					}).draw();
