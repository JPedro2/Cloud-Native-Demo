Intro

BiQ App Framework was created to provide the ability to quickly build very tailored apps using the Analytics API.

  • Full control over the look and feel with pure html and javascript.
  • Control over queries and updating the data on the fly.
  • Control over how to visualize the data and being able to interact with the visualizations.

Built on open source libraries. This allows you to easily integrate any libraries that you would like to use. We also wrap these libraries by using Chart and Component objects, which allows you to swap libraries if you like. Most Libraries have their Pros and Cons and we have chosen to pick a few libraries and mix and match based on the use case.


Main Technologies :
  • Node.js app using express views
  • C3.js for charts
  • Plotly.js for charts
  • JQuery

Component based, simple api and very customizable.

Generates the following panel with a Pie Chart backed by an ADQL query :

			
        new PieChartComponent({
            targetId:"pieChart",
            title:"Customers"
            query:"SELECT segments.userData.customerName, count(*) FROM transactions"
        }).draw(function(selection){
            alert(selection);
        })
