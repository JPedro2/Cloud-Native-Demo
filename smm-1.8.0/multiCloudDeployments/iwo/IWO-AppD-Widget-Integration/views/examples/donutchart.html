Donut Chart :

  • When working with the Donut Chart object you have to provide the data using the data property.
  • A wrapper for a C3.js Donut Chart (see c3.js donut chart )
  • You would use the Donut Chart when you want to have control over the data and the chart itself.

	var data = [
		["Customer Apple",1000],
		["Customer Amazon",2000],
		["Customer Cisco",3000],
		["Customer Facebook",4000],
	]
	
	new DonutChart({
		targetId:"chart",
		data:data,
		title:"Customers"
	}).draw(function(selection){
		alert(selection);
	})

Donut Chart Component

  • Adds an outer panel and you can provide a title. The title property can be dynamic and include some variable and or html
  • All chart components allow you to pass in an ADQL query instead of the data property. If you pass in both, the query would be executed first.
  • if you remove the data and query properties then the component will generate random or test data.
			
	new DonutChartComponent({
		targetId:"chartComponent",
		title:"Donut Chart Component"
		//query:"select customername,count(*) from transactions where application = 'test"
	}).draw(function(selection){
		alert(selection);
	})

Customizing Chart Options

  • To pass in C3 chart options provide a chartOptions property.
  • This will work for both Chart and ChartComponents.
			
	new DonutChartComponent({
		targetId:"chartComponentCustomized",
		title:"All Red",
		chartOptions:{
			size:{width:"200",height:"200"},
			color:{pattern: ['#FF0000']},
			legend: {show: false}
		}
	}).draw(function(selection){
		alert(selection);
	})
