Getting Started

The best way to get started using this framework is to :

  • Review the components documentation
  • Review the example apps and look at the source
  • Start editing the newdash.html and starting using the components
  • Edit the config.json and provide your api keys so you can start adding ADQL queries to your components
  • Review the examples

For help you can send an email to biq-apps [@] appdynamics.com
